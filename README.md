# Noted-auth

Сервис для авторизации в Noted (https://dnk33.com) с помощью keycloak

Known issues:

- Нет graceful shutdown
- Секреты лежат в .env вместо vault
- Полный игнор контекстов
- Нет PKCE
- Нет тестов
- Не настроены линтеры
- Нет сваггер-доки
- Не настроен CI/CD
- Нет обработчика /healthcheck
